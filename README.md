# my-bp-tracker

Progressive Web App (PWA) for tracking blood pressure and health readings with interactive charts and trend visualization.

## Features

- **BP Logging**: Record systolic/diastolic readings with timestamps
- **Interactive Charts**: Visualize BP trends over time using Chart.js with zoom/pan support
- **PWA**: Installable on mobile devices for native app-like experience
- **Offline Support**: Service worker enables offline access to logged data
- **Vercel Deployment**: API backend deployed on Vercel

## Tech Stack

- Vanilla JavaScript, HTML, CSS
- Chart.js (with zoom plugin)
- TailwindCSS
- Node.js API backend
- Vercel (hosting)

## Setup

1. Install dependencies: `npm install`
2. Run locally: `node app.js`
